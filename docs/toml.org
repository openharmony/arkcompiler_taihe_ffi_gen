#+TITLE: 项目配置文件

* 检查代码格式

使用ruff进行检查
#+BEGIN_SRC bash
poetry run ruff check [dirs]
#+END_SRC

使用black进行检查
#+BEGIN_SRC bash
poetry run black --check [dirs]
#+END_SRC

* 虚拟环境管理

1. 位于包含poetry的环境中克隆代码，可参考[[https://python-poetry.org/docs/cli#install][官方安装指南]]。

2.修改配置，使虚拟环境存储于compiler/.venv。
#+BEGIN_SRC bash
poetry config virtualenvs.in-project true
#+END_SRC

3.在compiler目录中创建并激活虚拟环境。
#+BEGIN_SRC bash
cd compiler
poetry shell
#+END_SRC

4.从当前项目中读取pyproject.toml文件，解析依赖项并安装。
#+BEGIN_SRC bash
poetry install
#+END_SRC
